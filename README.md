# Axa Project

***Api Axa***

### Setup

**clone**

```
git clone https://github.com/zercould/axa_project
```
**go to project folder**

```
cd axa_project
```
**install npm**

```
npm install
```
**start**

```
npm start
```

**testing**

```
npm test
```


*To test import the file 'axa.postman_collection.json' into postman*
