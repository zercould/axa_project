# Axa Project

***Api Axa***

### Setup

**clone**

```
git clone https://github.com/zercould/axa_project
```
**go to project folder**

```
cd axa_project
```
**install npm**

```
npm install
```
**start**

```
npm start
```

**testing**

*run the following command with the console on*

```
npm test
```
